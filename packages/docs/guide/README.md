## 功能概览

karin-plugin-kkk 是一个功能全面的 Karin 机器人插件，为中国主流社交媒体平台提供内容解析和自动推送能力。

### 核心能力

| 功能类别 | 说明 |
|---------|------|
| **内容解析** | 自动识别并解析抖音、B站、快手、小红书的分享链接，提取视频、图片、文本等内容 |
| **评论解析** | 支持多平台评论抓取，统一渲染为精美图片，保持视觉一致性 |
| **自动推送** | 订阅创作者账号，定时监控并推送新内容到指定群聊 |
| **精美渲染** | 基于 React SSR 的模板系统，提供高质量图片输出 |
| **灵活配置** | 支持画质偏好、体积限制、过滤策略等多维度配置 |
| **Web 管理** | 可视化配置界面，支持推送历史查看和数据库管理 |

### 平台支持矩阵

| 平台 | 内容解析 | 评论解析 | 自动推送 | 扫码登录 | 支持内容类型 |
|-----|---------|---------|---------|---------|------------|
| **抖音** | ✅ | ✅ | ✅ | ✅ | 视频、图集、直播、评论、动态 |
| **B站** | ✅ | ✅ | ✅ | ✅ | 视频、动态、番剧、直播、评论 |
| **快手** | ✅ | ✅ | ❌ | ❌ | 视频、评论 |
| **小红书** | ✅ | ✅ | ❌ | ❌ | 图文笔记、视频笔记、评论 |

### 技术特性

- **Monorepo 架构**：pnpm workspace 管理，模块化开发
- **TypeScript**：类型安全，开发体验优秀
- **React SSR**：服务端渲染，高性能图片生成
- **SQLite 数据库**：轻量级持久化存储
- **Puppeteer 渲染**：HTML 转图片，支持自定义精度
- **定时任务**：Cron 表达式配置，灵活的推送策略
- **代理支持**：解析库支持 HTTP/HTTPS 代理，适应各种网络环境
