本插件支持将各平台的评论区内容渲染为图片发送，方便在群聊中快速预览热评。

## 支持平台

该功能支持以下所有平台：

- **抖音**
- **B站**
- **快手**
- **小红书**

## 功能特性

### 1. 深度还原
- **表情包解析**：支持解析并显示平台自带的表情包（如抖音表情、B站小电视等）。
- **图片评论**：支持显示评论中附带的图片（包括 HEIC 格式自动转 JPG）。
- **用户高亮**：自动识别并高亮显示 @用户、#话题 等特殊文本。
- **粉丝勋章**：B站评论区支持显示用户的粉丝勋章和等级。

### 2. 智能排版
- **楼中楼**：支持渲染二级评论（回复），清晰展示对话结构。
- **置顶评论**：自动识别并置顶显示 UP 主/作者置顶的评论。
- **长文优化**：针对长篇大论的评论进行排版优化，确保阅读体验。

### 3. 配置选项

你可以在 WebUI 中针对每个平台单独配置：

- **解析数量**：控制渲染的评论条数（建议 10-20 条）。
- **二级评论限制**：控制楼中楼的显示数量，避免图片过长。
- **显示真实数量**：选择显示解析到的数量还是平台显示的“共 X 条评论”。
