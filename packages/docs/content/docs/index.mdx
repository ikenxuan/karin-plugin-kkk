karin-plugin-kkk 是 Karin 机器人的多平台内容解析与推送插件。

## 支持平台

| 平台 | 解析 | 评论 | 推送 | 扫码登录 |
|-----|:---:|:---:|:---:|:---:|
| 抖音 | ✓ | ✓ | ✓ | ✓ |
| B站 | ✓ | ✓ | ✓ | ✓ |
| 快手 | ✓ | ✓ | — | — |
| 小红书 | ✓ | ✓ | — | — |

## 核心功能

- **内容解析**：自动识别链接，返回视频信息、评论截图、视频文件
- **动态推送**：订阅创作者，定时推送新动态到群聊
- **精美渲染**：React 模板渲染，支持主题切换
