`karin-plugin-kkk` 是基于 [Karin](https://github.com/KarinJS/Karin) 框架开发的功能插件。

## 核心业务

本插件专注于**多平台短视频/图文内容的解析与推送**，旨在为群聊提供极致的媒体分享体验。

通过监听并解析用户发送的分享链接，插件会自动提取视频、图集及评论区热评，并使用精美的 React 模板渲染为图片发送，让群友无需跳转 APP 即可浏览完整内容。

## 支持平台

| 平台 | 解析能力 | 评论区渲染 | 动态推送 | 扫码登录 |
|-----|:---:|:---:|:---:|:---:|
| **抖音** | ✓ | ✓ | ✓ | ✓ |
| **B站** | ✓ | ✓ | ✓ | ✓ |
| **快手** | ✓ | ✓ | — | — |
| **小红书** | ✓ | ✓ | — | — |
