`karin-plugin-kkk` 係基於 <LinkPreview url="https://github.com/KarinJS/Karin" className="font-bold text-fd-primary hover:underline">Karin</LinkPreview> 框架開發嘅功能插件。

## 核心業務

本插件專注於**多平台短視頻/圖文內容嘅解析同推送**，旨在為群傾偈提供極致嘅媒體分享體驗。

通过監聽並解析用戶發送嘅分享連結，插件會自動提取影片、圖集同評論區熱評，並使用精美嘅 React 模板渲染成圖片發送，等群友無需跳轉 APP 即可瀏覽完整內容。
