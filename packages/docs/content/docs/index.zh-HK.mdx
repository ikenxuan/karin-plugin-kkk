`karin-plugin-kkk` 係基於 <LinkPreview url="https://github.com/KarinJS/Karin" className="font-bold text-fd-primary hover:underline">Karin</LinkPreview> 開發嘅多平台內容解析插件。

## 核心定位

佢嘅重點唔係「再發一次連結」，而係喺攞到平台數據之後，直接渲染成適合群聊閱讀同瀏覽嘅圖片內容。

- 自動識別抖音、B站、快手、小紅書分享連結
- 提取影片、圖文、熱評、動態內容並渲染成圖片
- 為抖音、B站提供動態推送、強制推送同統計能力
- 支持彈幕燒錄、Live Photo 兼容導出等增強能力

## 當前能力概覽

| 平台 | 自動解析 | 評論渲染 | 動態推送 | 掃碼登錄 | 彈幕燒錄 |
|-----|:---:|:---:|:---:|:---:|:---:|
| **抖音** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **B站** | ✓ | ✓ | ✓ | ✓ | ✓ |
| **快手** | ✓ | ✓ | — | — | — |
| **小紅書** | ✓ | ✓ | — | — | — |
