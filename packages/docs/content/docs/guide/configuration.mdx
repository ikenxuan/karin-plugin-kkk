配置文件位于插件数据目录，推荐通过 Karin WebUI 可视化修改。

## 通用配置

```yaml title="app.yaml"
# 渲染精度 50~200，建议 100
renderScale: 100

# 主题：0 自动、1 浅色、2 深色
Theme: 0

# 表情回应
EmojiReply: true

# Web 页面鉴权
webAuth: true
```

## 平台配置

<Tabs items={['抖音', 'B站', '快手', '小红书']}>
<Tab value="抖音">

```yaml title="douyin.yaml"
switch: true          # 解析开关
tip: false            # 解析提示

# 发送内容：info(信息) / comment(评论) / video(视频)
sendContent:
  - info
  - comment
  - video

numcomment: 5         # 评论数量
videoQuality: '4k'    # 画质：adapt / 540p / 720p / 1080p / 2k / 4k
maxAutoVideoSize: 50  # 自动画质时的体积上限(MB)
videoInfoMode: image  # 信息模式：text / image
```

**推送配置：**
```yaml
push:
  switch: true
  permission: master  # 权限：all / admin / master
  cron: '*/10 * * * *'  # 每10分钟检查一次
  parsedynamic: false   # 推送时是否解析作品
```

</Tab>

## 推送列表

使用命令 `#设置抖音推送 + 抖音号` 或 `#设置B站推送 + UID` 配置，也可手动编辑：

```yaml title="pushlist.yaml"
douyin:
  - short_id: "yuanshen_mihoyo"  # 抖音号
    group_id:
      - "群号:机器人账号"
    remark: "原神官方"

bilibili:
  - host_mid: 401742377  # UP主UID
    group_id:
      - "群号:机器人账号"
    remark: "某UP主"
```

<Callout type="info">
再次发送相同的订阅命令即可取消订阅。
</Callout>
