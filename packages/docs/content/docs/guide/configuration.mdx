## 推荐配置方式

本插件配置项多达近百项，涵盖各平台解析规则、推送策略、渲染参数等。为降低使用门槛并减少格式错误，**强烈推荐**通过 **Karin WebUI** 进行可视化配置。  
未来版本计划**逐步弃用可读写的 YAML 配置文件**，转而采用数据库存储用户配置，届时手动修改将不再生效。

<Callout type="warn" title="不推荐手动修改配置文件">
手动编辑配置文件极易出错，且部分配置存在联动关系，手动修改可能导致未知的异常。
</Callout>

### 如何进入配置页面

1. 启动 Karin
2. 访问 WebUI 控制台
3. 导航至 **插件管理** -> **插件配置** -> **karin-plugin-kkk**
