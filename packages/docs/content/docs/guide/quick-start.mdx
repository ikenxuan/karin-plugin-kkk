## 安装

<Steps>
<Step>

### 环境要求

- Node.js >= 22
- pnpm >= 9
- Karin >= 1.13.0

</Step>
<Step>

### 安装插件

推荐通过 Karin WebUI 插件市场安装，或使用命令：

```bash
pnpm add karin-plugin-kkk@latest -w
```

</Step>
<Step>

### 配置 Cookies

在 Karin WebUI → 插件配置 → karin-plugin-kkk 中填写各平台的 Cookies。

<Callout type="info">
Cookies 用于请求平台 API，不配置也能使用基础功能，但部分高级功能（如高画质视频、推送）需要登录态。
</Callout>

</Step>
</Steps>

## 基本使用

启动后会监听群聊消息，发送包含平台链接的消息，插件成功匹配以下链接后自动解析：

- 抖音：支持 `douyin.com` 与 `iesdouyin.com` 及其子域名 (`www`, `v`, `jx`, `m`, `jingxuan`)
- B站：支持 `bilibili.com`, `b23.tv`, `t.bilibili.com`, `bili2233.cn`，以及纯 **BV号** (`BV...`) 和 **av号** (`av...`)
- 快手：支持 `kuaishou.com`, `v.kuaishou.com`，以及APP分享的文本格式（如 `快手...快手`）
- 小红书：支持 `xiaohongshu.com`, `xhslink.com`
