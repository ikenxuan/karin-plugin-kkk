# @karin/cli

Karin 项目的构建 CLI 工具，用于统一管理多包构建并统计耗时。

## 功能特性

- 🚀 支持单包或多包构建
- ⏱️ 自动统计每个包的构建耗时
- 📊 显示总构建时间和详细统计
- 🎨 美观的终端输出
- ❌ 构建失败时自动停止后续构建

## 使用方法

```bash
# 构建单个包
kkk build core

# 构建多个包
kkk build core web

# 构建所有包
kkk build core docs template web
```

## 可用的构建目标

- `core` - 核心包
- `docs` - 文档包
- `template` - 模板包
- `web` - Web 包

## 输出示例

```
🚀 开始构建 2 个包: core, web

📦 构建 core...
✨ core 构建成功！耗时: 3.45s

📦 构建 web...
✨ web 构建成功！耗时: 2.18s

==================================================
📊 构建统计:
  ✓ core: 3.45s
  ✓ web: 2.18s

⏱️  总耗时: 5.63s
==================================================

🎉 所有包构建成功！
```
