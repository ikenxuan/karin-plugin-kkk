## 内容解析

### 自动解析

在群聊中发送包含平台链接的消息，插件会自动识别并返回：
- 视频/图片信息卡片
- 评论截图
- 视频文件（根据配置）

### 手动解析

当解析开关关闭时，回复包含链接的消息并发送 `#解析` 触发解析。

### 画质与体积

- `videoQuality: adapt` 时，会从高到低尝试各画质，直到体积小于 `maxAutoVideoSize`
- 设置固定画质（如 `1080p`）则直接下载该画质，不检查体积

## 动态推送

### 订阅管理

```
#设置抖音推送yuanshen_mihoyo   → 订阅抖音博主
#设置B站推送401742377          → 订阅B站UP主
```

再次发送相同命令即可取消订阅。

### 查看订阅

```
#抖音推送列表
#B站推送列表
```

### 强制推送

```
#抖音强制推送      → 当前群立即推送一次
#B站强制推送       → 当前群立即推送一次
```

<Callout type="info">
推送默认每 10 分钟检查一次，可在配置中修改 `cron` 表达式。
</Callout>

## 扫码登录

```
#抖音登录
#B站登录
```

扫码后自动保存 Cookies，用于获取高画质视频和推送功能。

## Web 管理

启用 `APIServer` 后可访问：

| 页面 | 路径 | 功能 |
|-----|------|------|
| 解析页面 | `/kkk/` | 在线解析链接 |
| 数据管理 | `/kkk/database` | 查看推送历史 |

端口取决于 `APIServerMount` 配置：
- 开启：使用 Karin HTTP 端口
- 关闭：使用 `APIServerPort` 配置的端口

## 权限说明

| 权限值 | 说明 |
|-------|------|
| `all` | 所有人 |
| `admin` | 管理员 |
| `master` | 机器人主人 |
| `group.owner` | 群主 |
| `group.admin` | 群管理员 |
