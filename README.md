> 提供对 Bot 的视频解析和动态推送功能，通过接口获取数据并渲染图片返回

</div>

---

## Feature

- **视频解析** - 自动解析抖音、B站、快手、小红书链接
- **动态推送** - 订阅博主/UP主，自动推送最新动态
- **扫码登录** - 支持扫码获取平台 Cookies
- **WebUI 配置** - 通过 Karin WebUI 轻松管理插件

## Quick Start

推荐通过 **Karin WebUI 插件市场** 安装，或使用命令：

```sh
pnpm add karin-plugin-kkk@latest -w
```

> 首次使用请务必查看文档了解配置方法
